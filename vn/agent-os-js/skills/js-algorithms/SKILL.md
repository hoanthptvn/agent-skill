# 🧮 Kỹ năng Thuật toán Front-End (Router)

> **Dành cho AI Agent:** Đây là tệp điều hướng. Không có thuật toán cụ thể ở đây. Hãy xác định nhu cầu của người dùng và đọc tệp tương ứng.

### Nhóm Thực chiến (Ngày nào cũng dùng)

Nằm trong thư mục `essential/`:

- `skills/js-algorithms/essential/animation-loop.md`: Xử lý `requestAnimationFrame`, tối ưu vòng lặp render.
- `skills/js-algorithms/essential/flip-sort.md`: Hiệu ứng kéo thả (Drag & Drop), Lọc danh sách mượt mà (FLIP).
- `skills/js-algorithms/essential/binary-search.md`: Áp dụng cho Virtual Scroll, Timeline Scrubbing.
- `skills/js-algorithms/essential/sliding-window.md`: Tính toán FPS, tốc độ cuộn (Scroll Velocity).
- `skills/js-algorithms/essential/two-pointers.md`: Chạm đa điểm (Pinch-to-zoom), thu gọn mảng in-place.
- `skills/js-algorithms/essential/frequency-counter.md`: Phân nhóm (Category Filter), đếm tần suất realtime.
- `skills/js-algorithms/essential/spatial-hash.md`: Tối ưu check hover chuột trên hàng chục ngàn phần tử.
- `skills/js-algorithms/essential/graph-bfs-dfs.md`: Hiệu ứng lan tỏa sóng (Ripple Effect) từ điểm click.
- `skills/js-algorithms/essential/pathfinding.md`: Tìm đường đi ngắn nhất (Dijkstra) cho web game/tương tác.

### Nhóm Chuyên sâu (Chỉ dùng cho WebGL / Framework)

Nằm trong thư mục `advanced/`:

- `skills/js-algorithms/advanced/radix-sort.md`: Sắp xếp Z-depth hàng ngàn particles ở 60fps.
- `skills/js-algorithms/advanced/binary-heap.md`: Hàng đợi ưu tiên (Priority Queue), Task Scheduler.
- `skills/js-algorithms/advanced/graph.md`: Quản lý Scene Graph (Three.js), State Machine cực độ phức tạp.
- `skills/js-algorithms/advanced/recursion.md`: Deep Clone, cấu trúc Tree View vô hạn.
